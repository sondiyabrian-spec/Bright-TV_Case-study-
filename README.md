# Bright-TV_Case-study-
Conducting an analysis for the CEO to understand the company's subscription base for this financial Year.

### Problem Statement 
The BrightTV has been under performing in the last 3 months (1st Quater) of the financial year i am task to perform a diagnostic on what caused the viewership drop. 

### Aim 
The aim of this project is to provide analysis with insght that will assist the COE to grow the company's subscription based for this finanacial year 

### Objectives 
The CEO of BrightTV aims to significantly increase the company's subscription base. To achieve this, the CVM team requires a deep understanding of user behavior and consumption patterns.
  
The analysis focuses on delivering a comprehensive understanding of BrightTV's viewership data, addressing the following key areas:
•	User and Usage Trends: Providing detailed insights into how users interact with BrightTV's content and overall usage patterns.
•	Factors Influencing Consumption: Identifying and analyzing the various factors that impact viewer consumption habits.
•	Content Recommendations: Suggesting specific content strategies to boost consumption, particularly on days identified with low viewership.
•	Initiatives for User Base Growth: Proposing strategic initiatives designed to further expand BrightTV's user base.

### Data Considerations
The dataset provided for this case study includes information on user profiles and viewer transactions for BrightTV. 
Key considerations for data handling include:
•	Time Zone Conversion: All times and dates in the dataset are supplied in Coordinated Universal Time (UTC) and must be converted to South African (SA) time for accurate analysis.
•	Session-Based Consumption: Consumption data is recorded per session, meaning each subscriber's viewing activity within a single session corresponds to one record.

### Data Preparation

1. Cleaned raw datasets (removed nulls, standardized fields)

2.Joined multiple tables using a common key

3.Created **age group segmentation**

4. Converted timestamps to **South African time (SAST)**
